NAVBHARAT-AI SETUP GUIDE

1. Install Node.js:
https://nodejs.org

2. Open project folder in terminal.

3. Run:
npm install

4. Create .env file:
Copy .env.example and rename to .env

5. Paste your Groq API key:
GROQ_API_KEY=your_key_here

6. Start server:
npm start

7. Open:
http://localhost:3000
